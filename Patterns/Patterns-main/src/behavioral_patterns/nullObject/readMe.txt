                                                Null Object
=======================================================================================================================
    Null Object - это объект с определенным нейтральным (null) поведением. Шаблон проектирования Nuul Object описывает
использование таких объектов и их поведение (или отсутствие такового).
=======================================================================================================================
    Стоит использовать, когда:
          - Объект требует взаимодействия с другими объектами. Null Object не устанавливает нового взаимодействия -
            он использует уже установленное взаимодействие;
          - Какие-то из взаимодействующих объектов должны бездействовать;
          - Требуется абстрагирование "общения" с объектами, имеющими null-значение;
=======================================================================================================================
                                    Преимущества и недостатки Null Object
    +
      * Отказ от условных операторов;
      * Уменьшает шанс исключений из-за нулевых указателей;
    -
      * Еще один новый класс;
=======================================================================================================================
